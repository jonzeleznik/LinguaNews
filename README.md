# LinguaNews

LinguaNews is a web app that scrapes news articles from various sources and translates them into Slovenian.

### Built with 
 - Tailwing: css
 - GO: logic
 - Templ: components that render HTML
 - Echo: Go web framework 
 - Colly: scrapes the web

### Getting started
1. `task install`
2. `task run`
